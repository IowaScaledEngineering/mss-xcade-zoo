# mss-xcade-zoo
A collection of custom track and signal configurations for the MSS-XCADE

For simplicity, these are all designed to build within the Arduino
environment.  It's far easier for the end user than trying to explain how to
set up the ESP IDF and all the complexities of a real build system.  

For all of these, the environment will need the ESP32 tools and the
mss-xcade control library installed.  Details on how to make that happen can
be found in the [Block Signal Custom Developer's
Guide](https://docs.iascaled.com/SimpleSig/Block%20Signal%20Custom/developer_guide/#getting-started).
